HKS Data Science Group

The HKS Data Science Group is a student run organization providing students of any skill level opportunities to improve their data science and machine learning skills with like-minded students.

Whether you're looking to learn programming skills for the first time or an experienced technician looking to work on projects for your portfolio, the HKS Data Science Group has something for you.
